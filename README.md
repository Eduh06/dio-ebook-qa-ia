# Projeto Ebook: QA & IA - A Evolução Indispensável 🚀

Este repositório contém um projeto do curso da [DIO]([https://www.dio.me](https://web.dio.me/track/universia-fundamentos-de-ia-generativa)/), focado na criação de um ebook profissional utilizando ferramentas de Inteligência Artificial.

## 📖 O Ebook
O ebook explora as mudanças na carreira de QA com a chegada da IA, desmistifica o fim da profissão e lista as novas habilidades necessárias para o mercado moderno.

- [Link para acessar o Ebook (PDF)](./ebook/seu-ebook.pdf)

## 🛠️ Tecnologias Utilizadas
* **ChatGPT**: Geração de conteúdo e estruturação de capítulos.
* **Midjourney**: Criação de artes visuais e capa.
* **Figma/Canva**: Diagramação e layout.
* **Markdown**: Organização da documentação.
